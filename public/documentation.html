MyFlix API

This incredible API will let YOU look up movies!

MyFlix API is a server-side application that client-side developers can use to provide up-to-date, comprehensive movie info to users.

Description URL HTTP Method Request Response
Return a List of ALL movies to the user /movies GET None JSON: a list of all movies in the database with title, description, genre, director, image URL, whether it’s featured or not.

Return data (description, genre, director, image URL, whether it’s featured or not) about a single movie by title to the user; use %20 for spaces /movies/:title GET None JSON: title, description, genre, director, image URL, whether it’s featured or not.
Example: See Above
Return data about a genre (description) by name (e.g., “Thriller”) /movies/genres/:genre GET None Description of a particular genre
Example:

Drama is a category of narrative fiction (or semi-fiction) intended to be more serious than humorous in tone.
Return data about a director (bio, birth year, death year) by name; use %20 for spaces /movies/directors/:name GET None Bio with Year of Birth and Year of Death (if deceased)
Example:

Orson Welles was an American actor, director, writer and producer who is remembered for his innovative work in radio, theatre and film.
Born: 1915
Died: 1985
